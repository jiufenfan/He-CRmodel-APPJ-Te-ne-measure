# Docs Agent Rule

- `ROADMAP.md` is forward-looking and currently only covers v0.1.0.
- `CHANGELOG.md` records completed changes only.
- `docs/devlog/` records active issues, decisions, known limitations, and handoff notes.
- `docs/devlog/ISSUE_DEPENDENCY_GRAPH.md` is the authoritative human-readable dependency gate, and `docs/devlog/issue_dependencies.yaml` is its compact agent source. Update both when issue ordering, blocked status, owner agent, or model recommendation changes.
- Important terminal failures must be summarized in `docs/devlog/ERROR_LOG.md`.
